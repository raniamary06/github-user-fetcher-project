# GitHub User Fetcher

A Python application that fetches GitHub user details using the GitHub API.

## Features
- Fetches Name, Username, Bio, Followers, Following, Public Repos, Account Creation Date
- Displays Top 5 Repositories by stars
- Handles invalid usernames gracefully

## How to run
- clone the repo and navigate to the correct folder
- create and activate a virtual environment
- create .env file and paste github token
- run the main.py
