# GitHub User Fetcher

A simple Python command-line application that fetches and displays GitHub user profile details using the public GitHub API.

## Features

* Retrieves a user's Name, Username, Bio, Followers, Following, Public Repositories, and Account Creation Date.
* Handles HTTP requests using the Python `requests` library.
* Gracefully handles 404 errors for nonexistent users.

## Prerequisites

* Python 3.x
* Git

## Installation and Setup

1. Clone the repository to your local machine.
2. Navigate to the project directory:
```bash
cd Athul-PP

```


3. (Optional but recommended) Create and activate a virtual environment.
4. Install the required dependencies:
```bash
pip install -r requirements.txt

```



## Usage

Run the script using Python:

```bash
python fetcher.py

```

When prompted, enter a valid GitHub username to view their profile statistics printed directly to the console.
